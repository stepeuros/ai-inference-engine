# AI Inference Engine

A small **matrix multiplication engine** in C with plans for future assembly optimizations.  
This project demonstrates how to build a simple AI inference core using **C**, optimized for low-level computation.

---

## Features

- Pure C **matrix multiplication**  
- Cross-platform compilation with **Clang/LLVM**  
- Easy to extend with **assembly-level optimizations**  
- Fully documented and ready for GitHub showcase
